# Guardrails - FlowLint WebLinter

## Rules

### G1: Never commit to main
- **Trigger:** `git commit` on main branch
- **Instruction:** Create feature branch
- **Discovered:** Iteration 0

### G2: Test in browser
- **Trigger:** Editor changes
- **Instruction:** Test in actual browser
- **Discovered:** Iteration 0

### G3: Mobile responsive
- **Trigger:** UI changes
- **Instruction:** Test on mobile devices
- **Discovered:** Iteration 0
